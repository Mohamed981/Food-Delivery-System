# Deployment
Open project directory and type 
```
$ docker compose up -d
```
or open Backend folder in another terminal and type
```
$ npm install 
$ npm run start
```
and open Frontend folder in a terminal and type
```
$ npm install
$ ng serve
```  
Then go to http://localhost:4200
